PostgreSQL for Ruby on Rails / Ubuntu 12.04
=====

## Setup

* `sudo apt-get install postgresql libpq-dev`
* `sudo -u postgres psql postgres`
* `\password postgres`
* `sudo -u postgres createdb mydatabase`
